## 🚀 Next Steps

### 1. Customize Your Content

Edit `src/pages/index.astro` to personalize:

- **Your name** and title in the hero section
- **About section** content and skills
- **Project descriptions** and technologies
- **Contact links** (email, GitHub, LinkedIn)

### 2. Update Contact Information

Replace the placeholder contact information:
- Email: `your.email@example.com`
- GitHub: `https://github.com/kuryaki`
- LinkedIn: `https://linkedin.com/in/yourprofile`

### 3. Add Your Projects

Update the projects section with your actual work:
- Project names and descriptions
- Technologies used
- Links to live demos or repositories

### 4. Deploy to GitHub Pages

Follow the deployment guide in `DEPLOYMENT.md`:

1. Push your code to GitHub
2. Enable GitHub Pages in repository settings
3. Set source to "GitHub Actions"
4. Your site will automatically deploy!

## 🛠️ Development

### Local Development
```bash
npm run dev
```
Visit `http://localhost:4321` to see your site locally.

### Building for Production
```bash
npm run build
```
The built files will be in the `dist/` directory.

## 🎯 Customization Tips

### Colors and Styling
The website uses a purple gradient theme. You can customize:
- Background gradients in the CSS
- Button colors and hover effects
- Typography and spacing

### Adding Pages
To add more pages:
1. Create new `.astro` files in `src/pages/`
2. Add navigation links in the header
3. The routing is automatic based on file names

### Adding Images
Place images in the `public/` directory and reference them as `/image-name.jpg`
